## Floral Palette

Floral Palette is a simple and elegant flower pot and planter showcase website developed using HTML and CSS. The website allows visitors to explore decorative flower pots, learn about the brand, browse products, and contact the store.

## Project Overview

Floral Palette is designed to provide users with an attractive and user-friendly platform to discover beautiful planters and gardening accessories. The website focuses on aesthetics, simplicity, and easy navigation.

## Features

Responsive navigation bar

Attractive homepage with floral background

About Us section introducing the brand

Product gallery displaying flower pots with prices

Contact form for customer inquiries

Footer with contact information and social media links

Clean and simple user interface

## How to Run the Project

1. Clone the repository:
git clone https://github.com/your-username/floral-palette.git
2. Open the project folder
3. Open index.html in your browser.
OR
4. Use VS Code Live Server:
Install Live Server extension.
Right-click on index.html.
Click Open with Live Server.
